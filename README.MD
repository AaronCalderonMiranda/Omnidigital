# 🎮 OmniDigital

OmniDigital es una plataforma web ficticia inspirada en servicios digitales de videojuegos.  
Simula una tienda y servicio de suscripción para juegos, mostrando un catálogo destacado, planes de membresía y secciones informativas.

Este proyecto fue creado con fines educativos para practicar diseño frontend y estructura visual de una plataforma gaming moderna.

---

## 📌 Secciones del sitio

- Inicio con carrusel de juegos destacados
- Catálogo de juegos populares
- Planes de suscripción:
  - OmniDigital Essential
  - OmniDigital Extra
  - OmniDigital Deluxe
- Barra de navegación:
  - Juegos
  - Servicios
  - Noticias
  - Tienda
  - Asistencia

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap

---

## 🎯 Objetivo del proyecto

- Practicar maquetación web
- Simular una plataforma de videojuegos tipo PlayStation / Xbox
- Mejorar habilidades en frontend
- Trabajar componentes visuales como cards, botones, grids y secciones dinámicas
